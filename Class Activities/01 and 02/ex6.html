The CSS code required to style tags is like this .assignments {
font-family: times new roman;
font-size: 10pt;
font-weight: bold;
color: purple;
}
Then when you are ready to style the assigments, you reference it like this:
< p class="assignments">Assignments< /p>" It would be extemely time consuming for each teacher to have their own styles. Because of this, it is preferable to have predetermined styles made in CSS so that it carries over throughout your document and also to give the HTML document a polished look and uniformity.
